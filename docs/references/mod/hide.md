# hide

> Hides a node from the animator. (one-liner style supported)

⚠️ This command is used to hide a node from the animation interface. Any keyable attributes will no longer be keyable once hidden.

## Example

**Inline syntax**
```yml
[mod]
hide: rig::roots.0
```

**Expanded syntax**
```yml
[mod]
hide:
  rig::roots.0
```
