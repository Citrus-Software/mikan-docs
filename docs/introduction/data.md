# Training Materials & Resources

:::note
This section is currently a Work in Progress. We are actively populating our libraries with more videos, scenes, and blueprints. Be sure to check back regularly for new updates and learning resources!
:::

### 📂 Practice Scenes & Assets

To follow along with the tutorials, we provide a dedicated folder containing all the necessary Maya scenes, meshes, and reference files.

- [Download Training Files (Google Drive)](https://drive.google.com/drive/folders/1nyRrfkHPoHY0XDg8Jt3YCKOhdY3X9BH9?usp=sharing)
    - Tutorial Assets: Raw meshes and starting scenes.
    - Reference Rigs: Finished examples to study hierarchy and module settings.

### 📺 Video Tutorials

These videos serve as visual companions to our step-by-step documentation. Each tutorial in this playlist is designed to complement the instructions found in the [Guides](../guides/intro) section, providing a real-time look at the workflow in Maya.

- [Official YouTube Playlist: Mikan Guides](https://www.youtube.com/watch?v=vPpMPfOGmBc&list=PLSb5n2G2Iu2M22f2QPdhQ5koLU1ha6MdQ)

### 💻 Blueprint Examples

For technical users looking to automate or study complex setups, we will maintain a collection of Blueprint files on GitHub.

- [Mikan Example Blueprints (GitHub)](https://github.com/Citrus-Software/mikan/tree/main/examples)
    - Usage: Use these to see how we structure multi-limb characters or specific creature setups.
    - Contribution: Feel free to explore the repository to see how Mikan handles data under the hood.
